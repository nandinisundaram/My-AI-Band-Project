# My-AI-Band-Project
Final project for the Building AI course: An AI assisted digital band to create music the way you want to.

## Summary
Ever wanted to create music but you didnt know the details of how to? This AI digital band will help you create music of your own.

## Background
This AI app will help you create music with fundamental knowledge. We will work on the project to create music with zero knowledge too. Problems addressed: The longing to listen to the kind of music you've always wanted to, have a certain amount of control over it with access to the instruments digitally. This problem may not be prevalent but some long to create music the way they want to hear it. My motivation is simply: creating music. Lets make it easier to create music that has the power to heal the world.

## How is it used?
It can be used by musicians to compose music. Or those longing to create music.
